## The Core Thesis

The AI supercycle rests on a chain of assumptions:

```
Scaling LLMs → AGI capability gains
AGI gains → massive enterprise revenue
Enterprise revenue → funds OpenAI compute bills
OpenAI bills → funds Oracle's debt repayments
Oracle debt → funded Nvidia GPU purchases
```

If **any link in this chain breaks**, the entire structure is at risk. Evidence gathered here suggests the first link — that scaling reliably produces AGI-level capability — is already weakening.
